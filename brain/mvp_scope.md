# MVP Scope

The MVP includes:

- SUPER_ADMIN panel (basic)
- Academy creation
- User invitations
- Membership system
- Teams management
- Player registration
- Player-parent relationship
- Training sessions
- Attendance tracking
- Player evaluations
- Parent dashboard (view progress)

Not included:

- Payments
- Competitions
- Scouting
- Advanced analytics
- Multi-academy interactions

Core goal:

Allow academies to operate digitally and allow parents to track player development.
